Как использовать StyleBase
Полное руководство по работе с базой стилей.
01
Выберите стиль
Используйте поиск или выпадающий фильтр по категориям. Переключатель «Все / Базовые / Сообщество» позволяет быстро ограничить источник. Нажмите карточку или «Просмотр» для деталей.
02
Скачайте CSS
Нажмите «Скачать CSS» — регистрация не нужна.
/* glassmorphism.css */
.glass-card { backdrop-filter: blur(20px); }
03
Подключите к проекту
Поместите файл в папку проекта и подключите в HTML.
<link rel="stylesheet" href="styles/glassmorphism.css">
04
Загрузите свой стиль
Зарегистрируйтесь и нажмите «+ Загрузить». CSS-анализатор автоматически определит тип стиля и построит превью.
